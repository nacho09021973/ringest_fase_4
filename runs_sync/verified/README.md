# Verified runs

This directory contains verified analysis outputs used for Paper 2.

The downstream statistical outputs were generated from frozen input tables using the scripts in `scripts/paper2/`. External catalog metadata were treated as frozen inputs before downstream analysis. Checksums are recorded in each run directory.

These artefacts support reproducibility of the tabulated analyses. They are not, by themselves, evidence for new physics.
